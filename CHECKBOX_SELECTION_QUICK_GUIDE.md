# Quick Reference: Checkbox Faculty Selection

## ?? How to Use (Admin)

### Assigning Faculty to a Subject:

1. **Navigate** to Subject Assignments page
2. **Click** "Assign Faculty" button on any subject card
3. **Search** (optional) - Type name or email to filter
4. **Select** - Click checkboxes for faculty you want
5. **Verify** - Check the selection counter
6. **Submit** - Click "Assign Faculty" button

---

## ? Key Features at a Glance

| Feature | How to Use |
|---------|-----------|
| **Select Faculty** | Click checkbox next to name |
| **Deselect Faculty** | Click checked checkbox again |
| **Search Faculty** | Type in search box at top |
| **View Selected Count** | Look at bottom of modal |
| **Clear Search** | Delete text from search box |
| **Submit Assignment** | Click "Assign Faculty" button |

---

## ?? Quick Tips

### ? DO:
- Click checkboxes to select multiple faculty
- Use search to find faculty quickly
- Check the counter to verify your selections
- Review the list before submitting

### ? DON'T:
- Don't need to hold Ctrl/Cmd anymore!
- Don't worry about losing selections
- Don't click outside modal while selecting

## ?? Troubleshooting

### Problem: Can't find a faculty member
**Solution:** Use the search box - type their name or email

### Problem: Selected wrong faculty
**Solution:** Click the checkbox again to deselect

### Problem: Don't know how many selected
**Solution:** Look at the counter at the bottom: "X faculty member(s) selected"

### Problem: Search shows no results
**Solution:** Clear the search box or check spelling

## ?? Common Workflows

### Assign 1 Faculty:
```
1. Click "Assign Faculty"
2. Click checkbox for faculty
3. Click "Assign Faculty" button
```

### Assign Multiple Faculty:
```
1. Click "Assign Faculty"
2. Click checkbox for each faculty
3. Verify count matches what you want
4. Click "Assign Faculty" button
```

### Search & Assign:
```
1. Click "Assign Faculty"
2. Type faculty name in search
3. Click checkbox for result
4. Clear search to add more (optional)
5. Click "Assign Faculty" button
```

## ?? Important Notes

?? **Replacing Assignments:**
- Assigning new faculty **replaces** all current assignments for that subject
- You'll see a warning message before submitting

? **Success Message:**
- After successful assignment, you'll see: "Faculty assigned successfully!"
- Page will refresh to show updated assignments

? **Error Handling:**
- If something goes wrong, you'll see an error message
- Try again or contact support if error persists

## ? Quick Actions Reference

### Emergency Disable
```
1. Click [ON] toggle to switch OFF
2. Click [Save Changes]
3. Done! (2 seconds)

Result: Students blocked immediately
```

### Set Time Window
```
1. Enable [Use Time Schedule]
2. Set Start date/time
3. Set End date/time
4. Click [Save Changes]

Result: Students can only select during window
```

### Always Available
```
1. Enable [Faculty Selection]
2. Disable [Use Time Schedule]
3. Click [Save Changes]

Result: Students can always select
```

## ?? User Workflow

### Step 1: Select Filters
```
1. Choose Subject (optional)
2. Choose Faculty (optional)  
3. Choose Year (optional)
4. Choose Semester (optional)
```

### Step 2: Select Columns
```
? Check/Uncheck desired columns
?
Table updates in real-time
```

### Step 3: Generate Report
```
Click "Generate Report"
?
View results with selected columns
?
Enrollment Time shows with milliseconds
```

### Step 4: Export
```
Click "Export Excel" or "Export PDF"
?
Only selected columns are exported
?
Faculty Email is NOT included ?
?
Enrollment Time IS included with milliseconds ?
```

## ?? Key Features Highlighted

1. **Column Selection** ?
   - Interactive checkboxes
   - Real-time table updates
   - Affects both display and exports

2. **Enrollment Time** ??
   - Millisecond precision
   - Format: `YYYY-MM-DD HH:mm:ss.fff`
   - Sorted chronologically

3. **Faculty Email Removed** ???
   - Not displayed in table
   - Not included in Excel export
   - Not included in PDF export

4. **Dynamic Exports** ??
   - Excel: Only selected columns
   - PDF: Only selected columns
   - Maintains formatting and styling
